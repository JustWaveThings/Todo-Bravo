# Todo Bravo

Todo App featuring:

- Factory Functions
- ES6 Modules
- Webpack based on my own Webpack Template.
- localStorage API
- CSS variables or CSS Custom Properties
- Flexbox
- 100% Javascript DOM manipulation
- Semantic HTML
- No id pollution in the global namespace
